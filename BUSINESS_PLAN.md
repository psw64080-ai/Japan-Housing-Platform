## 프로젝트 개요

### 🎯 미션
**일본에서 외국인이 편안하게 살 수 있는 주택을 찾고, 지역 주민과 연결하는 원스탑 플랫폼 제공**

### 💡 핵심 가치 제안
1. **언어 장벽 해결** - AI 번역으로 일본어 미숙자도 쉽게 접근
2. **생활 정보 통합** - 주택 검색 + 지역 정보 + 커뮤니티 한곳에서
3. **신뢰성 확보** - 평가 시스템 + 안전한 계약으로 사기 방지
4. **문화 교류** - 외국인과 일본 주민의 자연스러운 만남

### 🌍 타겟 시장
- **1차**: 도쿄, 오사카, 교토, 요코하마 (대도시)
- **2차**: 후쿠오카, 나고야, 삿포로 (지방 도시)
- **최종**: 전국 모든 지역

### 👥 사용자
| 유형 | 특징 | 규모 |
|------|------|------|
| 외국인 세입자 | 언어 장벽, 문화 차이, 신뢰 필요 | ~200만명/연 |
| 일본 집주인 | 국제 거래 경험 부족, 수익 극대화 욕구 | ~50만명 |
| 일본 셰어하우스 주인 | 공동 생활 문화 확산 욕구 | ~10만명 |

## 기본 정보

### 📊 시장 기회
- 일본 내 외국인 거주자: **2024년 약 296만명** (전년대비 17.3% 증가)
- 연간 임차료 시장규모: **약 120조 엔**
- 셰어하우스 시장 성장률: **연 15-20%**

### 🔍 문제점 분석
| 문제 | 현 상황 | 우리 솔루션 |
|------|--------|----------|
| 언어 장벽 | 일본어만 지원하는 사이트 대다수 | AI 번역 + 다국어 UI |
| 신뢰 부족 | 사기 사례 증가 | 평가/인증 시스템 |
| 정보 분산 | 주택/지역정보 따로 찾음 | 통합 플랫폼 |
| 외로움 | 지역 커뮤니티 없음 | 이웃 네트워크 |

## 12가지 핵심 기능

### **1️⃣ 외국인 맞춤 주택 검색**
**설명**: 일본 주택 검색 포털의 국제화 버전

**상세 스펙**:
- 검색 필터
  - 가격대 (월 30,000~200,000엔)
  - 지역 (도시, 구, 역 근처 거리)
  - 룸타입 (1K, 1DK, 2K 등)
  - 편의시설 (에어컨, 욕조, WiFi 등)
  - 외국인 환영 여부
  - 애완동물 동반 가능 여부
  
- 매물 정보
  - 고화질 사진/360도 촬영
  - 상세 설명 (다국어)
  - 계약 조건 명시
  - 집주인 프로필
  
- 지도 기반 검색
  - 직관적 위치 파악
  - 주변 편의시설 마크
  - 역까지의 거리 표시

**개발 기간**: 8주  
**핵심 기술**: React, Google Maps API, Elasticsearch

### **2️⃣ 셰어하우스 플랫폼**
**설명**: 일본 주민과 외국인이 함께 사는 공동생활

**상세 스펙**:
- 셰어하우스 유형 분류
  - 일본식 전통 가옥
  - 현대 아파트
  - 게스트하우스
  
- 하우스 정보
  - 현재 거주자 프로필 (나이, 직업, 취미, 언어)
  - 공용 공간 (주방, 거실, 욕실)
  - 하우스 규칙 및 문화
  - 월별 행사 (요리, 영화 감상 등)
  
- 입주 신청
  - 자기소개서 작성
  - 오너 승인 프로세스
  - 트라이얼 기간 (1주일)

**개발 기간**: 6주  
**핵심 기술**: 신청 관리 시스템, 프로필 매칭

---

### **3️⃣ 지역별 생활 정보 통합**
**설명**: 주택 주변 모든 생활 정보를 한눈에

**상세 스펙**:
- **카테고리별 정보**
  - 편의점 (위치, 영업시간, 다국어 지원)
  - 병원 (진료과목, 영어 대응 여부, 예약 방법)
  - 학교 (초중고, 유학생 입학 절차)
  - 도서관 (개관시간, 다국어 자료)
  - 우체국 (송금, 세금 신청)
  - 경찰/119 (긴급 연락처)
  
- **대중교통 정보**
  - 근처 역 목록
  - 시간표 (실시간)
  - 노선도 (한국어/영어/중국어)
  - 교통카드 정보
  
- **생활 정보**
  - 쓰레기 버리는 법 (요일별)
  - 계절별 날씨 대비
  - 지진 대피소
  - 커뮤니티 센터
  
- **비용 정보**
  - 평균 물가
  - 공과금 (전기, 수도, 가스)
  - 이동 평균 임차료

**개발 기간**: 10주  
**핵심 기술**: Google Maps API, 크롤링, 데이터베이스

### **4️⃣ AI 번역/채팅 시스템**
**설명**: 실시간 AI 번역으로 언어 장벽 제거

**상세 스펙**:
- **다국어 지원**
  - 영어, 중국어(간/번체), 한국어, 베트남어, 필리핀어, 태국어
  - 자동 언어 감지
  
- **채팅 기능**
  - 1:1 채팅 (세입자 ↔ 집주인)
  - 그룹 채팅 (셰어하우스)
  - 실시간 번역 표시
  - 채팅 히스토리 저장 (3년)
  - 파일 공유 (계약서, 사진 등)
  
- **번역 정확도**
  - Google Translate API / DeepL API 활용
  - AI 학습으로 개선 (6개월 이상)
  - 용어 사전 (부동산 특화)
  
- **음성 메시지**
  - 음성 녹음 + 자동 번역 텍스트
  - 발음 가이드

**개발 기간**: 8주  
**핵심 기술**: WebSocket, Google Cloud Translation, AI 모델 파인튜닝

### **5️⃣ 계약서 자동화**
**설명**: 표준 계약서 템플릿으로 안전한 계약 체결

**상세 스펙**:
- **계약서 종류**
  - 주택 임차 표준 계약서
  - 셰어하우스 규칙서
  - 보증인 동의서
  
- **자동 생성 기능**
  - 기본 정보 입력 (임차인, 집주인, 임차기간, 임차료 등)
  - 자동 계약서 생성
  - 조건별 추가 조항 선택
  
- **다국어 계약서**
  - 일본어
  - 영어
  - 중국어
  - 한국어
  
- **전자 서명**
  - 각 당사자의 디지털 서명
  - 타임스탐프 기록
  - 법적 효력 검토 (법률가 상담)
  
- **체크리스트**
  - 필수 항목 자동 확인
  - 보증금, 월세, 중개료 명시
  - 해지 조건 확인
  - 원상복구 비용 협의

**핵심 기술**: PDF 생성, 전자 서명, 법무 컨설팅

**상세 스펙**:

  - 소통 능력

- **리뷰 게시**
  - 정성적 평가 (텍스트)
  - 사진 (입주/퇴거 시 상태)
  - 숨겨진 문제점 공개
  
- **프로필 신뢰도**
  - 누적 평점 표시
  - 리뷰 수 표시
  - "검증됨" 배지
  - 위반 이력 표시
  
- **사기 방지**
  - 부정 리뷰 감지 AI
  - 의심 거래 플래그
  - 신고 기능
  - 관리자 수동 검수
  
- **평가 인센티브**
  - 리뷰 작성 시 포인트 적립
  - 신뢰도 높을수록 우선 노출

**개발 기간**: 6주

- **서비스 종류**

### **8️⃣ 비용 계산기 + 재정 관리**
**설명**: 투명한 비용 계산 및 재정 관리 도구

**상세 스펙**:
- **초기 비용 계산기**
  - 월세 입력 → 자동 계산
    - 보증금 (월세 × 1-2개월)
    - 선금 (보통 월세 × 1개월)
    - 중개료 (월세 × 0.5-1개월)
    - 이사비
    - 가구/가전 구매비
  - 총액 합계 및 분납 계획
  
- **환율 변환**
  - 한국원 ↔ 엔화 실시간 환율
  - 월평균 환율로 계산 (변동성 완화)
  - 다양한 통화 지원
  
- **월별 생활비 계산**
  - 임차료
  - 공과금 (전기 ¥5,000~15,000, 수도 ¥4,000~6,000, 가스 ¥3,000~8,000)
  - 식비 (1인 평균 ¥25,000~40,000)
  - 교통비
  - 통신비
  - 기타 (미용, 엔터테인먼트 등)
  
- **예산 추적**
  - 월별 지출 기록
  - 카테고리별 분류
  - 차트 시각화
  - 목표금액 설정
  
- **저축 계획**
  - 월별 목표 저축액 계산
  - 투자 권고사항 (이자 높은 상품)
  - 환송 가이드 (본국으로 송금)

**개발 기간**: 8주  
**핵심 기술**: React Charts, 환율 API, 재정 관리 알고리즘

### **1️⃣1️⃣ 이웃 커뮤니티 기능**
**설명**: 지역 주민과 외국인의 자연스러운 만남

**상세 스펙**:
- **지역 커뮤니티 그룹**
  - 구/시 단위 자동 그룹 생성
  - 역 단위 소그룹
  - 직업/취미별 관심사 그룹
  
- **커뮤니티 채팅**
  - 지역 정보 공유
  - 추천 맛집/명소
  - 질문 & 답변
